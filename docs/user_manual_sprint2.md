# User Manual — Sprint 2

## 📌 Introduction
This manual covers all new features delivered in A2A-CRM Phase 1 Sprint 2 (Groups H–L). It is intended for CRM users, admins, and technical teams. All features are DPDP-compliant and extend the Sprint 1 foundation.

## 🆕 New Features (Group-wise)

### Group H — Marketing Automation & Attribution
- Landing Page Builder: Create and publish campaign landing pages for lead capture.
- Live Chat Widget: Embed chat on your website to auto-create leads.
- Walk-in Enquiry Kiosk: Register walk-in leads at events/campus.
- Multi-Touch Attribution: Track all lead touchpoints for accurate source reporting.
- Cost-per-Lead Tracking: Monitor campaign spend and cost per lead.
- Marketing Automation Engine: Visual workflow builder for nurture, triggers, A/B testing, drip, and reporting.

**How to use:**
- Access via Marketing > Automation or Attribution menus.
- Use the builder to create landing pages, chat widgets, or kiosk forms.
- Configure automation workflows with drag-and-drop UI.
- View attribution and cost reports in Analytics.

### Group I — AI & Advanced Scoring
- AI-Assisted Lead Scoring: AI-generated scores and rationale for leads.
- Custom Qualification Questionnaires: Configurable BANT-style forms.
- Predictive Churn Flag: Flags leads at risk of dropping off.
- Sentiment Analysis: Flags negative/urgent inbound messages.
- Conversational AI Chatbot: WhatsApp/web chatbot for FAQs, booking, escalation.
- Predictive Enrolment Forecasting: AI-powered projections by programme/batch.
- Anomaly Detection: Alerts for unusual drops in funnel metrics.
- AI-powered Nurture Journey Builder: AI suggests optimal nurture journeys.

**How to use:**
- View AI scores and churn flags on lead detail pages.
- Fill qualification questionnaires from the lead record.
- Access chatbot via WhatsApp or web chat.
- Review forecasts and anomaly alerts in Analytics.

### Group J — Telecalling & Gamification
- Power/Auto-Dialler: Automated outbound calling from CRM lists.
- Call Scripts: Guided, branching scripts for counsellors.
- Supervisor Call Monitoring: Listen, whisper, barge-in for QA/training.
- Counsellor Gamification: Leaderboards, badges, rewards for KPIs.
- Business Card Scanner (OCR): Scan cards to auto-create leads (mobile).
- Mobile Offline Mode: Work offline, sync when reconnected.
- Biometric Authentication: Fingerprint/face unlock for mobile app.

**How to use:**
- Access dialler and scripts from Telecalling menu.
- Supervisors use Monitor panel for live calls.
- View gamification dashboard for ranks and badges.
- Use mobile app for OCR, offline, and biometric features.

### Group K — Customisation & Advanced Analytics
- Custom Fields: Add institution-specific fields to leads/applications.
- Custom Report Builder: Drag-and-drop analytics, export to Excel/PDF.
- Scheduled Report Delivery: Auto-email reports on schedule.
- Workflow Template Library: Import pre-built automation templates.
- System Health Dashboard: Monitor API status, queues, errors, uptime.

**How to use:**
- Admins configure custom fields and workflows in Settings.
- Build and schedule reports in Analytics > Reports.
- Monitor system health in Admin dashboard.

### Group L — Integrations & Document Management
- DigiLocker Integration: Applicants upload verified docs via DigiLocker.
- Aadhaar eKYC: OTP-based identity verification for applicants.
- Alumni Module Bridge: Auto-create alumni records, track referrals.
- LMS Enrolment Trigger: Auto-enrol students in LMS on admission.
- Agent Commission Workflow: Automate commission calculation, approval, payout.
- Agent Bulk Communication: Bulk email/WhatsApp/SMS to agent network.

**How to use:**
- Applicants use DigiLocker/Aadhaar flows in portal.
- Admins view alumni/LMS status in student record.
- Finance manages agent commissions in Agent module.
- Bulk comms via Agent Management > Communication.

## 🧭 End-to-End User Flow
1. Lead captured via landing page, chat, kiosk, or import.
2. Multi-touch attribution and cost tracking auto-applied.
3. Marketing automation triggers nurture workflows.
4. AI scoring, churn, and sentiment analysis guide counsellor actions.
5. Telecalling, scripts, and gamification drive engagement.
6. Applications, documents, and payments processed.
7. ERP/LMS/Alumni integration on conversion.

## ❓ FAQs
- **Q:** How do I enable a new automation workflow?
  **A:** Go to Marketing > Automation, click 'New Workflow', and use the builder.
- **Q:** Where can I see AI scores and churn flags?
  **A:** On the lead detail page, under the 'Scoring' and 'Churn' tabs.
- **Q:** How do I scan a business card?
  **A:** Use the mobile app, tap 'Scan Card', and follow prompts.
- **Q:** How do I add a custom field?
  **A:** Admins go to Settings > Custom Fields.
- **Q:** How do I trigger DigiLocker upload?
  **A:** Applicants see the option in the document upload section of the portal.

- No PII is stored in logs or analytics exports.
- For detailed technical/implementation info, see group-wise docs (H–L).
