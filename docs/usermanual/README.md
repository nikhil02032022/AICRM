## About This Folder

The `/docs/usermanual/` directory contains end-user documentation for CRM features delivered in each sprint. Each manual covers:

- **Who** can use the feature (roles & permissions)
- **How** to use each screen step by step
- **How data flows** through the system
- **Troubleshooting** common issues
