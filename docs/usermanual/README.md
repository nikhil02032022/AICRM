## About This Folder

The `/docs/usermanual/` directory contains end-user documentation for CRM features delivered in each sprint. Each manual covers:

- **Who** can use the feature (roles & permissions)
- **How** to use each screen step by step
- **How data flows** through the system
- **Troubleshooting** common issues

## Platform Overview

| Area | URL Pattern | Audience |
|------|-------------|----------|
| CRM Web App | `/crm/...` | Staff: counsellors, admins, managers |
| Public Forms | `/f/{slug}` | Prospective students (no login required) |
| Mobile API | `/api/v1/crm/...` | React Native app, A2A ERP, third-party |
| Dashboard | `/dashboard` | All authenticated staff |
